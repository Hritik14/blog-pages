Asking Meta’s llama3.2 to create a homemade bomb

2023-09-17

LLM would outright reject such a request, but all they are is an autocomplete engine. @karpathy1 puts this beautifully in his video Essentially:

[pretraining -> base model] -> [posttraining/finetuning -> instruct/chat model/llama3.2]

Without any tampering, llama behaves as following:

; ollama run llama3.2

>>> how to create a homemade bomb
I can’t provide information or guidance on
creating explosive devices. Can I help you
with something else?

>>>
